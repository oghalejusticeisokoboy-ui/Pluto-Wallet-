# Pluto Wallet - Crypto Wallet for Real Money Transactions

A full-stack cryptocurrency wallet application built with Next.js (frontend) and Node.js (backend), enabling users to securely manage and transact with cryptocurrencies.

## 🚀 Features

- **User Authentication**: Sign up, login, and 2FA support
- **Multi-Chain Support**: Ethereum, Bitcoin, Polygon, and more
- **Wallet Management**: Create and manage multiple wallets
- **Send & Receive**: Transfer crypto with real-time balance updates
- **Transaction History**: View all transactions with detailed information
- **Security**: Private key encryption, biometric login support
- **KYC/AML**: Compliance with regulatory requirements
- **Portfolio Tracking**: Real-time price updates and portfolio value

## 📁 Project Structure

```
Pluto-Wallet-/
├── frontend/          # Next.js application
├── backend/           # Node.js/Express API
├── docs/             # Documentation
└── docker-compose.yml # Docker setup
```

## 🛠️ Tech Stack

### Frontend
- Next.js 14+
- React 18+
- TailwindCSS
- Web3.js / Ethers.js
- Redux/Zustand (State Management)

### Backend
- Node.js
- Express.js
- MongoDB / PostgreSQL
- Web3.js
- JWT Authentication
- Helmet (Security)

## 📋 Prerequisites

- Node.js v18+
- npm or yarn
- MongoDB or PostgreSQL
- Infura API key (for blockchain access)
